Lua 数据结构--Table(一)

Table

　　Table 是 lua 中唯一的数据结构。它是一个关联数组，也就是说，这个数组不仅仅以数字做索引，除了 nil 和 NaN 之外的所有 lua 值都可以做索引。同时，它也是异构数组，即数据的值也可以是不同的类型。Table 可被用于表示普通数组、序列、符号表、集合、记录、图、树等等。

Lua 数据结构--Udata

Udata

　　在 lua 中 userdata 有两种实现，LUA_TLIGHTUSERDATA 和 LUA_TUSERDATA。light userdata 使用一个 void* 指针实现，full userdata 使用 Udata 结构实现。userdata 用于 C 和 lua 之间进行结构数据交互。将 C 提供的方法导入 lua 中，可以很方便的操作 C 提供的结构。而 lua 只能通过提供的方法来操作 userdata。这种方式类似于，用 C/C++ 写好了积木，由 lua 将这些积木拼接成想要的样子，这是 lua 最常见的使用场景。

Lua 数据结构--TString

TString

　　TString 是 lua 中用于存储字符串的数据结构。lua 中字符串是不可变的字节序列，它对二进制友好，即不是以 ‘\0’ 作为结尾的，可以存储二进制数据的。

Lua 数据结构--TValue

TValue

　　TValue 是实现 Lua 弱数据类型的主要数据结构。所有关于 lua 数据值的操作都会涉及到 TValue。因此 TValue 贯穿了整个 Lua 的实现。

　　在 lua.h 中的宏定义列出了 lua 语法中所涉及的所有基础类型。但实际上 lua 在内部实现上会对这些基础数据类型进行更细粒度的划分，例如：LUA_TSTRING 在 lua 的实现层面会分成 LUA_TSHRSTR 和 LUA_TLNGSTR 。
